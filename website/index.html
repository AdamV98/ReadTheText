Read The Text!

Elisabeth Knipschildt, Ghada Laribi, Ricardo Carreon, Adam Varga

The goal of this project is to deblur an image to an extent, where the text in the image conceiled by blurring and noise can be read.

Visit the GitHub page of the project @AdamV98/ReadTheText

Inverse filter

In order to recover a sharp image from the observed blurred one, several methods were tackled in literature. We start with basic naive inverse filter by first assuming that the noise can be neglected thus the blur model is reduced to f = g * h.

Wiener filter

Because of the noise present on the image, the naive solution is not sufficient. The Wiener filter displays better stability than the inverse filtering. It has properties of a band-pass and is therefore even well-suited to deal with moderate noise.

Tikhonov method

Post-processing

Post-processing was utilized on the resulting image after appliying the Tikhonov method. This step is expected to enhance the filtered image. First, a sharpening filter was applied, then thresholding to reduce the background noise, and the last step was the usage of the contrast limited adaptive histogram equalization.
